## Example Coordinates

- **Frage (Question)**: C-2 = Column C (10% from left), Row 2 (5% from top)
- **Timer**: O-2 = Column O (70% from left), Row 2 (5% from top)
- **Antworten**: C-5 = Column C (10% from left), Row 5 (20% from top)
- **Leaderboard**: O-6 = Column O (70% from left), Row 6 (25% from top)
- **Joker Info**: C-13 = Column C (10% from left), Row 13 (60% from top)

## Size Conversions

For 1920x1080 resolution:
- **Small Question**: 400x100 px
- **Medium Question**: 800x150 px
- **Large Question**: 1200x200 px
- **XLarge Question**: 1600x250 px

For 1280x720 resolution (automatically scaled):
- **Medium Question**: 533x100 px (scaled by 0.667)
- All sizes scale proportionally!
