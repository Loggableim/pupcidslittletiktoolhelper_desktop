Chat Commands (GCCE Integration)

Requires Global Chat Command Engine (GCCE) plugin to be enabled. Viewers can trigger weather effects via chat commands.

Allow viewers to trigger weather effects using /weather, /weatherlist, and /weatherstop commands.
Use the permission system below to control who can use weather commands.
Allow users to specify effect intensity (0.0-1.0).
Allow users to specify effect duration (1-60 seconds).
Available Commands:
  • /weather <effect> - Trigger a weather effect (e.g., /weather rain)
  • /weatherlist - List all available effects
  • /weatherstop - Stop all effects (subscribers only)
