GCCE HUD Overlay

Configure the HUD overlay system for displaying text and images via chat commands

Note: This plugin requires the Global Chat Command Engine (GCCE) to be enabled. Add the overlay to OBS: /plugins/gcce-hud/overlay.html

Chat Commands
Available Commands:
  • /hudtext [duration] <text> - Display text on overlay
  • /hudimage [duration] <url> - Display image on overlay
  • /hudclear - Clear all HUD elements (moderator only)
