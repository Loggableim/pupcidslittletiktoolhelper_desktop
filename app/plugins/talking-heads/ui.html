⚠️ Kein API-Schlüssel konfiguriert!
Bitte konfigurieren Sie den SiliconFlow API-Schlüssel:
1. Gehen Sie zu Dashboard > Einstellungen > TTS API Keys
2. Geben Sie Ihren SiliconFlow API Key ein
3. Klicken Sie auf "Save TTS API Keys"
4. Klicken Sie hier auf "API testen" um die Verbindung zu prüfen
Der API-Schlüssel wird zentral verwaltet und mit anderen Plugins geteilt (TTS Fish Speech, StreamAlchemy).
ℹ️ SiliconFlow API
Das Plugin verwendet die global konfigurierte SiliconFlow API mit FLUX.1-schnell für schnelle Bildgenerierung. Der API-Schlüssel wird zentral in den Settings unter siliconflow_api_key gespeichert.
