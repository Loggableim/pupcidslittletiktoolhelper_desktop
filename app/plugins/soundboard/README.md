# Soundboard Plugin

## Overview

The Soundboard plugin provides a comprehensive sound management system for your TikTok LIVE streams.

## Features

- 100,000+ sounds from MyInstants
- Custom sound uploads
- Gift-to-sound mapping
- Event triggers (follow, subscribe, share)
- Volume control

## Getting Started

Navigate to the **Soundboard** section in the sidebar to:

1. Browse and add sounds
2. Map sounds to gifts
3. Configure event sounds
4. Test your soundboard
